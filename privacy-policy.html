YouTube Automation Bot — Privacy Policy

Effective Date: October 19, 2025
Last Updated: October 19, 2025

1. Overview

This Privacy Policy explains how YouTube Automation Bot (“the Bot”) collects, uses, and protects user data on Discord.

2. Data We Collect

  • Discord User, Server, and Channel IDs
  • YouTube video metadata (titles, links, thumbnails)
  • System logs for debugging

3. How We Use Data

We use data only to automate YouTube notifications, improve the Bot, and prevent abuse.

4. Data Sharing

No data is sold or traded. Data may pass through Discord or YouTube APIs as required for functionality.

5. Retention and Deletion

We store data only as long as needed for functionality. You can request deletion by removing the Bot or emailing us.

6. Security

We use reasonable technical measures to protect stored data but cannot guarantee absolute security.
